## Описание проекта
Этот репозиторий содержит решения лабораторных работ по дисциплине "Основы алгоритмизации и программирования". Каждая лабораторная работа находится в отдельной ветке Git для удобства организации и проверки.

## Структура проекта
- **Ветка `main`/`master`** - содержит базовый шаблон проекта и документацию
- **Ветка `lab1`** - первая лабораторная работа: "Знакомство со средой программирования и структурой проекта"
- Последующие лабораторные работы будут размещаться в соответствующих ветках (`lab2`, `lab3` и т.д.)

## Текущая работа
**Лабораторная работа №1:** Программы линейной структуры. Решение задач с использованием базовых конструкций языка C#.

## Технологии
- Язык программирования: C#
- Среда разработки: JetBrains Rider
- Система контроля версий: Git
